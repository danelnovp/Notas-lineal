Matriz Asociada a una Transformación Lineal

En esta sección se vera que cada transformación lineal T:A→B tiene asociada una matriz, tal que si se toma el vector de coordenadas de v∈A y se multiplica por esta matriz, entonces el resultado de la multiplicación sera el vector de coordenada de T(v)∈B.

Matriz en transformaciones reales

Primero se estudiara el caso real y luego se generalizara para cualquier matriz.

Teorema 1

Sea T una transformación lineal tal que T:Rn→Rm, entonces se puede encontrar una única matriz A∈Mm×n​(R) tal que T(v)=Av, donde v∈Rn.

Sea vi​ un vector canónico de Rn, con 1≤i≤n, donde i representa la posición del 1 en el vector, es decir:

vi​=(0,⋯,i-esima posicioˊn1​​,⋯,0)

Por este teorema sabemos que T esta determinado según se asigne los vectores de la base de Rn vectores de Rm. Sea w1​=T(v1​),w2​=T(v2​),⋯,wn​=T(vn​), donde cada wi​∈Rm y

wi​=​α1i​α2i​⋮αmi​​​

con αni​∈R, donde 1≤n≤m.

Sea A una matriz tal que sus columnas es cada uno de los vectores wi​:

A=[w1​​w2​​⋯​wn​​]=​α11​⋮αm1​​⋯⋯​α1n​⋮αmn​​​

Es claro que A∈Mm×n​(R). Sea vi​ un vector canónico de la base de Rn entonces si se multiplica este vector por la matriz se obtendrá como resultado el vector wi​

​α11​⋮αm1​​⋯⋯​α1n​⋮αmn​​​​0⋮1⋮0​​=​α1i​α2i​⋮αmi​​​=wi​

Aquí el 1 esta en la i-ésima fila de vi​.

Se puede escoger cualquier base de Rn y esto se seguirá cumpliendo, sea {v1​,v2​,⋯,vn​} una base no canónica de Rn, suponga que se tiene una transformación lineal tal que T(vi​)=wi​, donde i=1,⋯,n, al ser una base se puede encontrar las combinaciones lineales que nos den como resultado los vectores de las base canónica, es decir:

αij​∈Rα11​v1​+⋯+αn1​vn​=v1′​=(1,0,⋯,0)α12​v1​+⋯+αn2​vn​=v2′​=(0,1,⋯,0)⋮α1n​v1​+⋯+αnn​vn​=vn′​=(0,0,⋯,1)

Teorema 2

Sea V,W espacios vectoriales con dimensión n y m respectivamente, sea T∈L(V,W), sea B1​={v1​,⋯vn​} una base de V y sea B2​={w1​,⋯,wm​} una base para W, entonces existe una única matriz AT​ de m×n tal que: (T(v))B2​=AT​(v)B1​​

Recordemos que la notación (v)B​ se refiere al vector de coordenadas de v respecto a la base B, en caso de no acordarse revise Vectores coordenadas.

Tomemos T(vi​)=yi​ donde vi​∈B1​ ahora, yi​ es una combinación lineal de los vectores de la base B2​, es decir:

yi​=a1i​w1​+⋯+ami​wi​

por lo tanto el vector de coordenadas de vi​ serán los escalares de la combinación lineal anterior.

(yi​)B2​=​a1i​⋮ami​​​

Con esto se definira AT​ como la matriz de m×n tal que sus columnas con los vectores de coordenadas de cada yi​:

AT​=[(y1​)B2​​(y2​)B2​​⋯​(yn​)B2​​]=​a11​⋮am1​​⋯⋯​a1n​⋮amn​​​

Sea v∈V un vector arbitrario tal que (v)B1​=(c1​,⋯,cn​). Si se multiplica por AT​ se obtiene:

At​(v)B1​​=​a11​⋮am1​​⋯⋯​a1n​⋮amn​​​​c1​⋮cn​​​=​a11​c1​⋮am1​c1​​++​⋯⋯​++​a1n​cn​⋮amn​cn​​​=c1​​a11​⋮am1​​​+⋯+cn​​a1n​⋮amn​​​=c1​(y1​)B2​+c2​(y2​)B2​+⋯+cn​(yn​)B2​​

Se sabe que T(v)​=T(c1​v1​+⋯+cn​vn​)=c1​T(v1​)+⋯+cn​T(vn​)=c1​y1​+⋯+cn​yn​​ de esta manera se tiene que T(v)B2​​=(c1​y1​+⋯+cn​yn​)B2​=c1​(y1​)B2​+⋯+cn​(yn​)B2​​

Tomando el ejemplo anterior podemos tomar la base B1​={(2,1),(1,0)} y B2​={(1,0,0),(0,1,0),(0,0,1)}, luego tenemos que:

T(2,1)=(1,1,0)T(1,0)=(0,2,1)
